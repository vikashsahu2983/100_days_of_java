Create a Java program that generates a Participation Certificate for students who take part in a college event.
The program should collect all important details and print a properly formatted certificate at the end.

Required Inputs

Your program must take the following details from the user:

Student Name

Roll Number

Course / Department

Year / Semester

Event Name

Event Date (DD-MM-YYYY)

Event Venue

Organizer Name

Faculty Coordinator Name

Certificate Issue Date

(You may add extra fields like Student ID, College Name, or Event Type based on your preference.)

Program Requirements

Display the College Name at the top of the certificate.

After collecting all inputs, generate a neatly formatted participation certificate.

The certificate must contain:

Student details

Event details

Thank you note

Organizer & faculty coordinator names

Format should look clean and professional, similar to a real certificate.

Sample Input 1

Student Name: Rohan Sharma
Roll Number: 21CSE102
Course: B.Tech CSE
Year/Semester: 2nd Year
Event Name: Coding Hackathon
Event Date: 15-11-2025
Event Venue: Seminar Hall
Organizer Name: Tech Club
Faculty Coordinator: Prof. Meena Gupta
Certificate Issue Date: 16-11-2025

Sample Output 1

================= PARTICIPATION CERTIFICATE =================
ABC College of Engineering

This is to certify that Rohan Sharma (Roll No: 21CSE102)
from B.Tech CSE, 2nd Year has successfully participated in
the Coding Hackathon held on 15-11-2025 at the Seminar Hall.

Organized by: Tech Club
Faculty Coordinator: Prof. Meena Gupta

Certificate Issued On: 16-11-2025

Thank you for your participation.
