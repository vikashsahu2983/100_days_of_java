Create a Java program that generates an official donation receipt for the Happy Earth Charity Fund.
The program should collect complete donor information, donation details, and display a formatted receipt at the end.

Program Requirements

Your program must take the following inputs from the donor:

Donor Name

Age

Mobile Number

Email

Address

Donation Category

Education

Health

Food

Environment

Animal Care

Other

Payment Mode

UPI

Cash

Credit Card

Debit Card

Net Banking

Donation Amount

Donation Date (DD-MM-YYYY)

After collecting all inputs, the program should print a clean and formatted donation receipt that includes:

Charity Name (at top and bottom)

Donor details

Donation details

Personalized thank-you message

Sample Input 1

Enter Donor Name: Rohan Sharma
Enter Age: 25
Enter Mobile Number: 9876543210
Enter Email: rohan@gmail.com

Enter Address: Delhi, India
Enter Donation Category: Education
Enter Payment Mode: UPI
Enter Donation Amount: 1000
Enter Donation Date: 22-11-2025
