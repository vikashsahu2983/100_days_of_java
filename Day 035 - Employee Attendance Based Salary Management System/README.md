Day 035 – Program – Employee Attendance Based Salary Management System

Problem Statement:
Ek Java program banaiye jo employee ki basic details lekar unki monthly salary calculate kare. Program ko user se ye inputs lene hain:

Employee Name

Age

Designation (ex: Manager, Developer, Clerk, etc.)

Basic Monthly Salary

Total Working Days in Month

Present Days

Rules:

Absent Days = Total Days – Present Days

Daily Salary = Basic Salary / Total Working Days

Final Salary = Daily Salary * Present Days

Salary Deduction = Basic Salary – Final Salary

Output me ye sab clearly print karna hai:

Employee Name

Age

Designation

Present Days

Absent Days

Final Salary

Total Deduction

Sample Input 1

Name: Riya
Age: 28
Designation: Developer
Basic Salary: 40000
Total Days: 30
Present Days: 27

Sample Output 1

Employee Name: Riya
Age: 28
Designation: Developer
Present Days: 27
Absent Days: 3
Final Salary: 36000
Deduction: 4000

Sample Input 2

Name: Karan
Age: 35
Designation: Manager
Basic Salary: 60000
Total Days: 30
Present Days: 29

Sample Output 2

Employee Name: Karan
Age: 35
Designation: Manager
Present Days: 29
Absent Days: 1
Final Salary: 58000
Deduction: 2000
