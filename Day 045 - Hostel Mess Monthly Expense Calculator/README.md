You need to build a Java program that calculates the monthly mess expenses for a hostel student. The program should take daily meal usage details, compute the total cost, apply discounts when applicable, and generate the final monthly bill.

User Input Requirements

Student Name

Room Number

Number of Days Stayed in the Month

Daily Meal Count

Breakfast Count

Lunch Count

Dinner Count

Meal Cost Rules

Breakfast → 30 per meal

Lunch → 60 per meal

Dinner → 80 per meal

Program Requirements

Calculate total cost for each meal type.

Calculate the gross monthly mess expense.

Apply a 5% discount if the student stayed for more than 25 days.

Display a proper monthly bill with all totals and the final payable amount.

Sample Input 1

Student Name: Arjun Kumar
Room Number: 204
Days Stayed: 28
Breakfast Count per Day: 1
Lunch Count per Day: 1
Dinner Count per Day: 1

Sample Output 1

Hostel Mess Monthly Bill
Student: Arjun Kumar
Room: 204
Days Stayed: 28
Breakfast Total: 840
Lunch Total: 1680
Dinner Total: 2240
Gross Amount: 4760
Discount: 238
Net Payable: 4522
